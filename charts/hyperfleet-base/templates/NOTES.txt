To check the status:
  kubectl get pods -n {{ .Release.Namespace }}

To view API logs:
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name=hyperfleet-api

  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name=sentinel
